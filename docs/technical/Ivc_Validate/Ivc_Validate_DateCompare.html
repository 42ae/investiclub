Ivc_Validate_DateCompare

Compares two dates

Usage: $element->addValidator(new Ivc_Validate_DateCompare('startdate')); // exact match $element->addValidator(new Ivc_Validate_DateCompare('startdate', 'enddate')); // between dates $element->addValidator(new Ivc_Validate_DateCompare('startdate', '<')); // not later $element->addValidator(new Ivc_Validate_DateCompare('startdate', '>')); // not earlier $element->addValidator(new Ivc_Validate_DateCompare('startdate', true, 'm-d-Y')); // not later and specified element has value in date format m-d-Y
