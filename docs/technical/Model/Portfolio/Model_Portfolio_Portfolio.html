Model_Portfolio_Portfolio

Description
Description | Methods (details)

Portfolio class take care of the club's transaction.

It communicates with the treasury to register buy and sell actions. The portfolio calculate transaction, transaction statistics and curency exchanges. It also commumicate with the stock manager to register new stocks and get lasts updates.
