Model_Treasury_Treasury

Description
Description | Methods (details)

The Treasury is the class that handle the club contability.

It store every operation made by the club like buy a stock, add a cotisation or calcul a member departure share ...

The treasury contain two set of functions : - Functions that calcul the treasury with active / passive (all function call : "funct_") - Functions to add / remove or edit data

The treasury also save a balance sheet every month. The treasury use Ivc_Cache to store every month that way it don't have to query the database and calculate everything all the time.
